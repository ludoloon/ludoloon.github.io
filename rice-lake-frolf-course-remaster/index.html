Rice Lake Frolf Course Remaster

2024/07/23

I've recreated the WITC Disc Golf Course, available to play on either regular screens or in VR. Includes self-hosted multiplayer.

This is a proof of concept prototype that I was inspired to make after learning how to use BlenderGIS to download map data. It's still got bugs, but you can play the course like you're walking around it in real life.

I'm gonna be real busy for the foreseeable future so it's probably gonna stay like this for a while, unless any bugs are relatively easy fixes. That said when I can get back to it, I'd love to add more if there's interest.

Overall I learned a lot about Godot, 3D modeling and gamedev from this project.
